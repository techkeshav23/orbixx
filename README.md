# Orbixx Fitness

Online Zumba & fitness platform exclusively for women. Live classes, expert trainers, and a supportive community — all from home.

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
